
# Temporal and Social Analysis of Prescription Patterns for Contested Diagnostic Conditions

This project investigates temporal and socioeconomic patterns in NHS England prescribing data for ADHD medications and antidepressants (2020-2025). Using data fetched programmatically from the OpenPrescribing API, we explore whether prescription rates reflect genuine population health variation or systematic biases in the healthcare system - examining longitudinal trends, socioeconomic gradients, and regional variation across Integrated Care Boards (ICBs).
